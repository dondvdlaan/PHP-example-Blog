PHP-example-Blog

Initial set-up for a blog in PHP, which comprises of 2 pages: index.php 
and dashboard.php.
At start-up the existing blogs are displayed and user can select the category to 
be shown. After login, user is conducted to the dashboard, which consists of 
2 forms: category form and blog form.

Summary:
1. PHP Back-End 
2. HTML CSS Front-End
3. MySQL DB for user, account and category tables
